# centerfascia_application

서강대학교 캡스톤 디자인2 무면허 자동차 개발부 센터페시아 파트입니다.




[![CENTERFASCIA](https://img.youtube.com/vi/ILSYkSeJOCI/0.jpg)](https://www.youtube.com/watch?v=ILSYkSeJOCI "CENTERFASCIA")
시연영상 (이미지 클릭)

제공 기능 : 
  1. 카메라 : 사용자의 얼굴을 찍은 뒤, 서버에 전송하여 유저를 특정한 뒤, 해당 유저의 정보를 사용할 수 있도록 합니다.
  2. 메인메뉴 : 차량 내부의 기능들을 조작, 사전에 유저 어플리케이션에 저장해둔 플레이리스트, 그리고 관심장소의 페이지를 볼 수 있도록 합니다.
  3. HW Control : 차량 내부의 의자, 사이드 / 백미러, 무드등과 같은 하드웨어적인 기물들을 조작할 수 있도록 합니다.
  4. GPS : PIN번호를 입력하여 유저를 확실하게 특정을 한 뒤, 유저 어플리케이션에 등록해 두었던 관심장소를 불러옵니다.
  5. PLAYLIST : 4와 마찬가지로 유저 어플리케이션에 등록해 두었던 음악 플레이리스트를 불러옵니다.
